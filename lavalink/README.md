# Local Lavalink

This folder runs a local Lavalink node for Dro Tunes.

## Requirements

- Java 17 or higher

## Setup

1. Run:

```powershell
.\download-lavalink.ps1
```

2. Start Lavalink:

```powershell
.\start-lavalink.ps1
```

3. In another terminal, start the bot:

```powershell
cd ..
.\start-local.ps1
```
